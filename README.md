# LM7805 Voltage Regulator PCB

## Description
This project is a 12V to 5V linear voltage regulator designed using LM7805 in KiCad.

## Features
- Input: 7V–35V DC
- Output: 5V regulated
- Load: ~100mA
- Thermal copper pour for heat dissipation
- Screw terminal connectors

## Tools Used
- KiCad
- PCB Design & DRC

## Learning
- PCB layout and grounding
- Thermal management
- Limitations of SPICE simulation in KiCad

## Files
- Schematic (.kicad_sch)
- PCB Layout (.kicad_pcb)
- Project file (.kicad_pro)
